# Deepfake-Detection-Project

AI/ML-based system for detecting face swap deepfake images. The project analyzes uploaded facial images using image processing and machine learning techniques to identify whether the image is real or fake. The system helps improve digital media verification and reduce misuse of manipulated content.

# Features
- Deepfake image detection
- Real vs Fake prediction
- User-friendly interface
- Image upload and processing
- AI/ML-based prediction model

# Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Machine Learning

# Project Modules
- Frontend Interface
- Backend Processing
- Deepfake Detection Model
- Image Processing Module

# Project Workflow
1. User uploads image
2. Image gets processed
3. AI/ML model analyzes image
4. System predicts result
5. Output displayed as Real or Fake
